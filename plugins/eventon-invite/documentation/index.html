Intel EventOn Invite Plugin

Installation & Set up

Upload and install the Plugin as you would a typical WordPress plugin

Eventon Invite utilizes shortcodes to send the settings to the plugin.

Step 1: Create a Thank You Page

Create a New Page and title it 'Thank You'.

Copy and paste the shortcode below into the content area of the Thank You page. Change the parameters as needed

Sample shortcode:

[invite file="http://yourdomain.com/wp-content/plugins/eventon-invite/sample.json" button-text="Invite Guests" post-url="http://yourdomain.com/"]

Parameters

file: the location of the JSON file containing the Groups to invite
button-text: the text within the Invite button on the Thank You page
post-url: the URL to POST the invite data to

** When testing, pleas ensure that the post-url is a real URL. It can be the homepage for testing purposes

Step 2: Modify Create Event Shortcode

Open the Create An Event page (/form) in WordPress. Edit the shortcode found in the content.

Parameters rdir and rlink need to be added.

Sample shortcode:

[add_evo_submission_form lightbox="no" msub="no" rdir="yes" rlink="http://54.193.71.60/thank-you/" header="Create an Event"]

Parameters

rdir: set to "yes" (to activate the redirect)
rlink: enter the full URL to the Thank You page you created in Step 1

Add TimeZone Field To Frontend Form

In WordPress go to myEventON -> Action User

Select the Form Fields Tab

Scroll to bottom and enable 'Timezone'. Drag the Timezone bar to where you would liek it located in th Front End form

Save Changes


Testing

To view the Invite data, create an event and submit the invite. You can then view the Console for the data


The POST array will include:
event_id,event_title ,evcal_subtitle ,event_details ,ics_url ,group ,event_location ,event_time ,evcal_organizer,evcal_type
